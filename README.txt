Application de gestion de signatures 
2015

Prérequis : 

1 - Installer la dernière version de Nodejs à partir du site officiel
2 - Install Bower and Grunt CLI : 'npm install bower' - 'npm install grunt'
3 - Go the the root of 'MarocCloudSign'
4 - CLI : 'grunt serve'
5 - Launch the mongoDB Server : "sudo mongod"
6-  Launch the REST Interface : "java -server -jar restheart.jar"
